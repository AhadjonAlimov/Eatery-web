# Description: This project includes:
- ReactJS,
- Redux Reducer,

- Tailwind CSS,
- Fully responsive,
- Framer motion animations without affecting loading speed,
- Get - Post methods,
- Only admin can add new product.

---------------------------------------

To run this project follow next steps:

1. `npm install`
2. You need configure firebase.config.js
3. `npm start`
